# Panasonic Japan Kitchen Appliances

Home Assistant integration for Panasonic Japan kitchen appliances.

## Features

- Monitor electricity usage (kWh/month)
- Track cost reduction from energy-saving features
- View device status and operation mode
- Real-time updates

## Installation

Install via HACS or manually copy to `custom_components/panasonic_japan`.

## Configuration

Requires an Auth0 access token. See README.md for details.
